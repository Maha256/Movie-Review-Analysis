IMDB Movie Review Sentiment Analysis
==================================

A simple Streamlit application that predicts whether an IMDB
movie review is Positive or Negative using a trained Simple RNN
model built with TensorFlow (Keras).

Requirements
------------
- Python 3.10
- numpy
- tensorflow
- streamlit

Run the App
-----------
1. Activate your virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Start the app:
   streamlit run app.py

Note
----
Make sure `simple_rnn_model.h5` is present in the project folder.
